# Структура базы знаний (knowledge_base.md)

Файл `knowledge_base.md` — это Markdown-документ, который бот использует для ответов на вопросы клиентов. Структура основана на заголовках:

## Иерархия заголовков

| Уровень | Синтаксис | Назначение |
|---------|-----------|------------|
| Продукт | `# Продукт: Название` | Линейка товаров (например, ТВ-приставки) |
| Категория | `## Категория: Название` | Раздел (общие вопросы, покупка, настройка) |
| Подкатегория | `### Вопрос: Текст вопроса` | Конкретный вопрос |
| Ответ | Текст после `###` | Ответ на вопрос (до следующего `###`) |

## Пример

```markdown
# Продукт: ТВ-Приставки NEXT OS

## Категория: Общие вопросы

### Вопрос: Что такое ТВ-приставка NEXT?
Ответ: NEXT — это российский бренд умных ТВ-приставок...

### Вопрос: Какие версии существуют?
Ответ: Мы предлагаем NEXT Pro и NEXT One...
```

## Как обновить базу

1. Отредактируйте `knowledge_base.md` — добавьте новые товары, вопросы и ответы.
2. Пересоберите индекс Qdrant:
   ```bash
   docker compose exec bot python rebuild_qdrant_knowledge.py
   ```
3. Бот автоматически подхватит изменения при следующем запросе (перезапуск не нужен).

## Советы

- Пишите ответы чётко и по делу — бот ищет по смыслу.
- Добавляйте синонимы и разные формулировки вопросов, если клиенты спрашивают по-разному.
- Разделяйте блоки горизонтальной чертой `---` для читаемости.
